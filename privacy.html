隐私声明

最后更新日期：2025年08月09日

Apexi Building Inspections Pty Ltd（“我们”）重视您的隐私。本声明说明我们在网站（含 GitHub Pages 托管的下载页）上如何收集、使用、存储和保护您的个人信息。

我们收集的信息

  • 联系信息：姓名、电子邮箱、（可选）电话。
  • 技术数据：访问日期时间、浏览器类型、页面访问路径（仅用于统计与安全）。

我们如何使用信息

  • 向您发送《验房避坑清单》下载链接或相关资料。
  • 向您提供与验房服务相关的沟通与支持。
  • 改进我们的网站内容与用户体验。
  • 在您同意的情况下发送营销更新（您可随时取消订阅）。

处理与共享

我们可能使用第三方服务处理表单提交与网站托管，例如：

  • Formspree（表单投递服务）——用于接收您的表单提交。
  • GitHub Pages（静态托管）——用于托管我们的网站页面。

我们不会出售您的个人信息。仅在遵守法律或保护合法权益的必要情况下共享。

数据保留

我们仅在实现上述目的所需的期间内保留您的个人信息，或依据法律、税务与审计要求保留更长时间。

您的权利

  • 访问、更正或删除您的个人信息。
  • 撤回同意或反对我们出于特定目的处理个人信息。
  • 如对我们的处理方式有疑问或投诉，可随时联系我们。

信息安全

我们采取合理的技术与组织措施保护您的信息安全。但通过互联网传输的数据无法保证 100% 安全，请您理解并采取必要防护。

跨境传输

如果您的信息被发送至或存储在澳大利亚以外的国家/地区，我们会采取合理措施确保接收方提供与澳大利亚法律相当的保护水平。
